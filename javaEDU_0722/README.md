## 자바기초 1일차

### 1.디렉토리
```bash
root
 ㄴ bin
 ㄴ lib
 ㄴ src
    ㄴ 파일명.java
```
#### 1) bin 폴더 
- 자바 소스(.java) 파일을 컴파일한 결과로 생성되는 바이트코드(.class 파일)들이 저장되는 디렉터리.
- 소스코드(src>*.java) 컴파일하면 bin 폴더에 실행 가능한 .class 파일이 생김.

#### 2) lib 폴더
- 라이브러리 폴더

#### 3) src 폴더
- 소스코드 작업 관련 폴더


### 2. class (생성자)
```java
public class HelloWorld {
    // 변수(필드), 메서드(함수) 등을 정의할 수 있음
}
```
- 객체를 만들기 위한 그룹
- 내부에 변수, 메서드(함수) 등 정의함
- 해당 생성자 바탕으로 동작하는 객체를 만듬
